Phone Controlled Mouse & Keyboard (PROTOTYPE)

Rust, WinAPI, Cargo, Socket Programming, Multi-threading

Phone Controlled Mouse & Keyboard (PROTOTYPE), written in Rust, listens on a local socket and processes incoming JSON data to control mouse movements, clicks, scrolling, and typing. It is powered by asynchronous programming using tokio and utilizes the serde library for JSON deserialization.

Sand Simulation 2
Python, Pygame, Data Structures

Sand Simulation 2 is a particle-based physics simulation developed in Python using Pygame. The simulation features a variety of interactions, including: sand sinking to the bottom of water, oil floating on water, fire burning through oil, wood, and moss to create ash, and the transformation of water and lava into stone and steam, or obsidian and smoke, and more!
